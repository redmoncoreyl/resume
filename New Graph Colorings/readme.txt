Compile: g++ <C++ file> -o <executable> -lemon

Run: ./<executable> [params]

The executable will describe the parameters required if you
attempt to use incorrect parameters.

Read Report.pdf to understand the project.

File structure:
  SourceCode
    graphs - contains graphs and graph generators
    greedyAlgorithms - contains greedy algs for colorings
    properColoringLocalSearch - contains local search algs
    results - contains results of colorings
    weakColoringLocalSearch - contains local search algs
  readme.txt
  Report.pdf
